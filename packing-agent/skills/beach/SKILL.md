# Beach trip skill

Use this skill when the forecast `summary` is **hot** (or **warm**) or when the trip
`purpose` is **beach** — sun, sand, and swimming destinations. It covers the beach-
specific gear on top of any weather- or profile-driven items the agent adds.

Guidance: pack for sun and water — quick-drying clothing, swimwear, and skin/eye sun
protection. The weather tool already supplies the sun hat and sunscreen when it is hot;
this skill fills in the swim and beach-day items the forecast alone does not capture.

## Packing items
- swimwear :: clothing
- beach towel :: essentials
- flip-flops :: footwear
- sunglasses :: accessories
- after-sun lotion :: toiletries
- light cover-up :: clothing
