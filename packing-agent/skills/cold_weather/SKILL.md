# Cold-weather trip skill

Use this skill when the forecast `summary` is **cold** or **freezing** (snow, sub-10°C,
winter destinations). It covers the warmth layers and accessories a traveller needs on
top of any weather- or profile-driven items the agent adds.

Guidance: prioritise insulating layers you can stack, keep extremities (hands, head,
feet) covered, and protect skin from cold, dry air. The weather tool already supplies the
outer jacket and any waterproofing; this skill fills in the base/mid layers and
accessories that the forecast alone does not capture.

## Packing items
- thermal base layers :: clothing
- fleece mid-layer :: clothing
- insulated gloves :: accessories
- wool beanie :: accessories
- warm wool socks :: clothing
- scarf :: accessories
- lip balm :: toiletries
